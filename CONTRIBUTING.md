# Contributing to Awesome Pet Skills 🐾

## What We Accept

✅ AI agent skills focused on the human-pet relationship
✅ Skills that help preserve a pet's memory or personality
✅ Skills that improve pet care through data intelligence
✅ Open source or free-tier with a public GitHub repo
✅ Compatible with at least one agent platform

## What We Don't Accept

❌ Generic AI tools that aren't pet-specific
❌ Novelty/joke pet apps (fake translators, costume generators)
❌ Pet product recommendation engines or affiliate plays
❌ Skills without a working GitHub repository
❌ Anything disrespectful to the human-animal bond

## How to Submit

1. Fork this repository
2. Add your skill to the appropriate category in README.md
3. Use this format:

```
| [Skill Name](https://github.com/owner/repo) | What it does — be specific about what pet data goes in and what value comes out. | Platform1 · Platform2 |
```

4. Submit a PR with title: `Add [Skill Name]`
5. In your PR description, explain:
   - What pet data the skill works with
   - What it produces for the owner
   - Whether it's for living pets, passed pets, or both

## Suggesting New Categories

If your skill doesn't fit existing categories, propose a new one. Categories should describe a type of human-pet interaction, not a technical implementation.

## A Note on Sensitivity

Many people will find this list because they lost a pet. Please keep all descriptions, PR comments, and discussions respectful and empathetic. We're building tools for grief, memory, and love. Act accordingly.
